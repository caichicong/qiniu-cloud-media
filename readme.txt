上传图片到七牛云，并可以在编辑器里直接引用

== Description ==

为七牛云文件建立custom type，在编辑器界面点击按钮可以插入图片。

== Installation ==

下载插件并解压，上传至 `/wp-content/plugins/` 目录，然后启用插件
